# Mülakat Hazırlık Dokümanı

Bu doküman, Backend Engineer, Senior Backend Engineer, Software Architect ve Team Lead rollerine hazırlanmak için oluşturulmuş teknik mülakat bilgi tabanıdır.

## Hedeflenen Roller

- **Backend Engineer**: Temel backend yetkinlikleri, .NET ekosistemi, veritabanı yönetimi
- **Senior Backend Engineer**: Dağıtık sistemler, performans optimizasyonu, mimari kararlar
- **Software Architect**: Sistem tasarımı, teknoloji seçimi, scalability stratejileri
- **Team Lead**: Teknik liderlik, mentorluk, takım yönetimi

## Çalışma Mantığı

Bu doküman şu mantıkla organize edilmiştir:

1. **Teknik Konular**: Mülakatta sorulabilecek teknik konuların derinlemesine açıklamaları
2. **Proje Anlatımları**: Gerçek projelerden örneklerle desteklenen hikayeler
3. **Mülakat Soruları**: Teknik ve davranışsal sorular için hazırlıklı cevaplar
4. **Eksik Konular**: Çalışılması gereken alanlar ve 7 günlük çalışma planı
5. **Mülakat Notları**: Her mülakat sonrası not alma şablonu

## Teknik Konular

Teknik konular bölümünde şu başlıklar yer alır:

- .NET ve ASP.NET Core
- Microservices mimarisi
- Distributed systems
- Kafka ve Message Queue
- Redis cache
- PostgreSQL
- System Design
- Observability
- Security ve Authentication
- Performance ve Scalability

Her konu için mülakatçının neyi test ettiği, kısa ve detaylı cevaplar, proje örnekleri ve sık sorulan sorular yer alır.

## Proje Anlatımları

Gerçek projelerden örneklerle desteklenen hikayeler:

- Real-Time Market Data sistemi
- Backend services
- WebSocket distribution
- Redis cache implementasyonu
- PostgreSQL performance optimizasyonu

Her proje için problem, çözüm, teknik detaylar ve mülakatta anlatım stratejisi belirtilmiştir.

## Sık Gelen Sorular

Teknik ve davranışsal sorular için hazırlıklı cevaplar:

- **Teknik Sorular**: Message queue, retry, timeout, circuit breaker, DLQ, Redis, PostgreSQL, WebSocket, auth, performance analysis, error handling, distributed consistency, idempotency, BackgroundService
- **Davranışsal Sorular**: Kendini tanıtma, iş değiştirme nedeni, güçlü/zayıf yönler, problem çözme, takım çalışması, kariyer hedefleri
- **Şirkete Sorulacak Sorular**: Teknik kültür, code review, mentorluk, deployment, monitoring, test kültürü, kariyer gelişimi

## Eksik Konular

Çalışılması gereken alanlar ve 7 günlük çalışma planı:

- Circuit breaker detayları
- Outbox pattern
- Saga pattern
- OAuth2 / OpenID Connect
- Kubernetes temel kavramları
- Distributed tracing
- Advanced PostgreSQL indexing
- System design kapasite hesabı
- Load balancing
- Rate limiting
- API Gateway

## Mülakat Sonrası Notlar

Her mülakat sonrası not almak için şablon:

- Şirket ve pozisyon bilgisi
- Sorulan teknik ve davranışsal sorular
- Zorlandığım ve iyi cevapladığım sorular
- Eksik kaldığım konular
- Şirket hakkındaki izlenim
- Sonraki mülakata kadar çalışılacaklar
- Genel değerlendirme
