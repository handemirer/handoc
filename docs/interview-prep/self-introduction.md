# Kendini Tanıtma

Bu bölümde mülakatın "Kendini tanıtır mısın?" sorusu için hazırlıklı cevap şablonları yer alır.

## Genel Yapı

Kendini tanıtma cevabı şu yapıda olmalıdır:

1. **Kimim**: İsim, mevcut rol, deneyim süresi
2. **Ne yapıyorum**: Mevcut işin özeti, ana sorumluluklar
3. **Neyi seviyorum**: Teknoloji alanındaki ilgi alanları
4. **Hedefim**: Neden buradayım, ne arıyorum

## Şablon

```
Merhaba, ben [İsim]. [X] yıllık deneyime sahibim ve şu anda [Şirket]'te [Rol] olarak çalışıyorum.

Günlük olarak [ana sorumluluklar - 2-3 cümle] üzerinde çalışıyorum. Son zamanlarda [özel proje veya teknoloji] üzerinde yoğunlaştım.

Özellikle [teknoloji alanı - örn: distributed systems, real-time data processing] konularına ilgim var ve bu alanda derinleşmek istiyorum.

Şu anda [rol - örn: Senior Backend Engineer] pozisyonuna geçiş yapmayı hedefliyorum çünkü [neden - örn: daha büyük sistemler tasarlamak, liderlik deneyimi kazanmak].
```

## Örnek Cevap

Merhaba, ben Han. 5 yıllık deneyime sahibim ve şu anda bir fintech şirketinde Backend Engineer olarak çalışıyorum.

Günlük olarak gerçek zamanlı market data sistemi geliştiriyor, Kafka üzerinden yüksek hacimli veri akışı yönetiyor ve Redis cache ile PostgreSQL performans optimizasyonu yapıyorum. Son zamanlarda WebSocket ile real-time data dağıtımı ve distributed tracing implementasyonu üzerinde yoğunlaştım.

Özellikle distributed systems, microservices mimarisi ve high-performance backend sistemleri konularına ilgim var ve bu alanda derinleşmek istiyorum.

Şu anda Senior Backend Engineer pozisyonuna geçiş yapmayı hedefliyorum çünkü daha büyük sistemler tasarlamak, teknik liderlik deneyimi kazanmak ve daha karmaşık scalability problemleri çözmek istiyorum.

## Dikkat Edilecek Noktalar

- **Kısa ve net**: 1-2 dakika sürmeli
- **Teknik odak**: Teknik yetkinlikleri vurgula
- **Doğal**: Ezber gibi değil, akıcı olmalı
- **İlgili**: Pozisyona uygun yetkinlikleri vurgula
- **Güncel**: Son çalışmalardan bahset

## Farklı Rollere Göre Uyarlama

### Backend Engineer için:
- Temel backend yetkinliklerini vurgula
- .NET, ASP.NET Core, veritabanı deneyimi
- API geliştirme deneyimi

### Senior Backend Engineer için:
- Distributed systems deneyimi
- Performance optimizasyonu
- Mimari kararlar
- Code review ve mentorluk

### Software Architect için:
- Sistem tasarımı deneyimi
- Teknoloji seçimi süreçleri
- Scalability stratejileri
- Cross-team collaboration

### Team Lead için:
- Teknik liderlik deneyimi
- Takım yönetimi
- Mentorluk
- Project management

## Tekrar Kontrol Listesi

- [ ] Kendi şablonunu oluştur
- [ ] Farklı roller için uyarla
- [ ] 1-2 dakikada söyleyebileceğini test et
- [ ] Doğal akışı sağla
