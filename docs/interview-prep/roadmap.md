# Mülakat Hazırlık Yol Haritası

Bu yol haritası Backend Engineer, Senior Backend Engineer, Software Architect ve Team Lead rollerine hazırlanmak için düzenli bir çalışma planı sunar.

## Hedef

**Kısa Vadeli (1-3 ay):**
- Senior Backend Engineer rolüne geçiş
- Teknik konuları derinleştirme
- System design case'lerini çözmek
- Mock interview'ler yapmak

**Orta Vadeli (3-6 ay):**
- Software Architect veya Team Lead rolü
- Cross-team architectural decisions
- Mentorship ve leadership deneyimi

**Uzun Vadeli (6+ ay):**
- Principal Engineer veya Engineering Manager
- Organizational impact
- Strategic technical decisions

## 7 Günlük Plan

### Gün 1: Temel Hazırlık
- Kendini tanıtma cevabını hazırla
- Rol beklentilerini anla
- Proje hikayelerini gözden geçir
- Mock interview yap (kendi kendine)

### Gün 2: .NET ve ASP.NET Core
- .NET technical topics
- ASP.NET Core middleware pipeline
- Dependency injection
- Async/await ve threading
- Kendi projelerinden örnekler

### Gün 3: Distributed Systems ve Kafka
- Kafka / Message Queue
- Distributed systems concepts
- Offset management, idempotency
- Retry, circuit breaker, DLQ
- Real-time data processing

### Gün 4: Database ve Performance
- PostgreSQL indexing ve optimization
- Redis caching strategies
- Performance tuning
- Query optimization
- Connection pooling

### Gün 5: System Design
- System design principles
- Scalability strategies
- Capacity planning
- System design case çöz
- Trade-off analysis

### Gün 6: Behavioral ve Şirket Soruları
- Behavioral questions (STAR format)
- Şirkete sorulacak sorular
- Mock behavioral interview
- Feedback al

### Gün 7: Full Mock Interview ve İnceleme
- Full mock interview (teknik + behavioral)
- Zorlandığın soruları tespit et
- Eksik konuları çalış
- Final review

## 30 Günlük Plan

**Hafta 1-2: Temel Konular**
- .NET, ASP.NET Core, PostgreSQL, Redis
- Teknik soruları çalış
- Proje hikayelerini hazırla

**Hafta 3-4: Advanced Konular**
- Distributed systems, Kafka
- System design case'leri
- Performance optimization

**Hafta 5-6: Practice**
- Mock interview'ler (haftada 2-3)
- System design case çözümleri
- Behavioral interview practice

**Hafta 7-8: Final Hazırlık**
- Tüm konuları tekrar
- Eksikleri kapat
- Real interview'lara başla

## Günlük Çalışma Rutini

**Sabah (1 saat):**
- Teknik konu çalışma
- Kısa cevapları ezberle

**Öğleden Sonra (1 saat):**
- System design case çöz
- Coding practice

**Akşam (30 dakika):**
- Behavioral sorular
- Şirket soruları
- Mock interview

**Toplam Günlük:** 2.5 saat
