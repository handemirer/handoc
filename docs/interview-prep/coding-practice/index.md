# Coding Practice

Bu bölümde C# / .NET odaklı coding practice konuları yer alır. Mülakatlarda sorulan coding soruları için hazırlık.

## Konular

- [.NET Common Patterns](dotnet-common-patterns.md) - DI, Options Pattern, Repository, BackgroundService, HttpClientFactory
- [LINQ](linq.md) - Select, Where, GroupBy, Any, All, First vs FirstOrDefault, deferred execution
- [Async / Await](async-await.md) - async/await, Task, cancellation token, deadlock riski, ConfigureAwait
- [Collections](collections.md) - List, Dictionary, HashSet, Queue, ConcurrentDictionary

## Her Konunun İçeriği

Her coding practice sayfasında şu bölümler yer alır:

1. **Mülakatta Neden Sorulur?** - Konunun önemi
2. **Bilmem Gerekenler** - Temel kavramlar
3. **Örnek Sorular** - Mülakat soruları
4. **Örnek Kodlar** - C# kod örnekleri
5. **Sık Yapılan Hatalar** - Yaygın hatalar
6. **Kırmızı Bayrak Cevaplar** - Kaçınılması gereken cevaplar

## Çalışma Stratejisi

1. Her konunun temel kavramlarını öğren
2. Örnek kodları yaz ve çalıştır
3. Mülakat sorularını cevapla
4. Sık yapılan hataları anla
5. Pratik yap (LeetCode, HackerRank)
