# Mülakat Soruları

Bu bölümde teknik ve davranışsal sorular için hazırlıklı cevaplar yer altır.

## Soru Kategorileri

- [Teknik Sorular](technical-questions.md) - Backend, distributed systems, database, system design
- [Davranışsal Sorular](behavioral-questions.md) - Kendini tanıtma, güçlü/zayıf yönler, problem çözme
- [Şirkete Sorulacak Sorular](company-questions.md) - Teknik kültür, mentorluk, deployment, monitoring

## Teknik Sorular Formatı

Her teknik soru için:

- **Mülakatçı Neyi Ölçüyor?** - Sorunun amacı
- **Kısa Cevap** - 1-2 cümlelik özet
- **Detaylı Cevap** - Teknik detaylar
- **Kendi Projemden Örnek** - Gerçek proje örneği
- **Dikkat Edilecek Nokta** - Önemli noktalar

## Davranışsal Sorular Formatı

Her davranışsal soru için:

- **STAR Format** - Situation, Task, Action, Result
- **Doğal ve profesyonel cevap**
- **Ezber gibi değil**

## Şirkete Sorulacak Sorular

Kategorilere ayrılmış sorular:

- Teknik kültür
- Code review
- Mentorluk
- Deployment süreci
- Monitoring / observability
- Test kültürü
- Ürün-yazılım ilişkisi
- Kariyer gelişimi
- İlk 3 ay beklentisi
- Teknik borç yönetimi

## Çalışma Stratejisi

1. Teknik soruların kısa cevaplarını ezberle
2. Detaylı cevapları anla
3. Kendi proje örneklerini ekle
4. Davranışsal sorular için STAR formatı hazırla
5. Şirkete sorulacak soruları listele
